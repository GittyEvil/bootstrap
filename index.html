Bootstrap Uppgift


HTML Element

exempel på html element skulle vara t.ex. p element, h1,h2. Används för att skapa hemsidor.

HTML generellt

står för Hyper text markup language, vilket är ett språk för att skapa hemsidor.Alla rubriker, text, rutor och annat är gjort av html.

ID

attributet id används för att ge ett element ett id vilket blir helt unikt för hela dokumentet och används då för att lättare styla om en specifik sak istället för alla elementen samtidigt.

Class

specifierar en speciell egenskap eller mer för ett element. Används för att underlätta arbete specifikt för att styla om en hemsida enklare och inte ändra för t.ex. alla p element.

Meta

syftar på att du placerar något bakom, mellan, vid sidan eller över. Är ett prefix för olika diskussioner, inom datasammanhang så används det för att beskriva data av olika slag.

Responsive

är när du använder html och css för att ändra storlek, gömma undan, minska, öka storlek eller mer för att göra så att det ser bra ut på skärmen.

HTTP

Hypertext transfer protocol, är ett protokoll som beskriver kommunikation mellan enheter på ett nätverk. Detta används på hemsidor, i början av url. http har olika funktioner som , -get , -put, -post, -session och -delete. Som gör olika saker såsom att du ber om åtkomst för en hemsida, Frågar webbservern om information på hemsidan och de svarar med html sidan.

DNS

Domain name server, sätter namn på ip adresser, så istället för att du skriver in en ip adress för att gå in på en sida så skriver du t.ex. google.com.

CDN

Content delivery network vilket är ett protokoll som syftar på olika internet servrar som jobbar tillsammans för att ge ett så bra och snabbt innehåll från internet fort.

Compressed data

innebär att data kodas om så att det tar mindre utrymme än tidigare. målet är att ta bort all onödig data. Trycka ihop datan så den blir kompakt.

Local Bootstrap

Handlar egentligen om att webbsidor som använder local bootstrap använder bara en enda server vilket gör det segt medans t.ex. CDN kan skicka information fortare över internet till skillnad från Local bootstrap.

JQUERY

Är ett javascript(js) bibliotek som är skapt för att förenkla html och css. Generellt gör det enklare att använda js. Den har många funktioner som man i vanligtvis behöver skriva massa rader för.

Flexbox

är en css modell som gör att du kan lättare ändra utseendet på hemsidan lätt, som även anpassar utseendet beroende på skärmstorlek. Flexbox använder sig av containrar och items.

Grid

är en typ av layout för css, som skapar komplexa layouter för din webbläsare på ett enklare sätt och mer konsekvent sätt.

Code Snippets

är lathundar som gör det enklare att skriva upprepade loopar och andra upprepande statements. Skapat för att underlätta för programmeraren.

WASM

Webb Assembly är ett nytt programmeringsspråk som är gjort för att köra olika koder på olika språk på webben vid en snabb hastighet, vilket leder till att vissa appar kan köras som förut inte kunde göra det. Den jobbar samtidigt/ i hand med javascript, man kan även kombinera web assembly med javascript:ens api vilket bara förbättrar prestandan,kraft och då använder sig av javascript:ens flexibla del.

Chromium

är en open source browser vilket är när alla användare kan göra ändringar och dela med andra för att få en bättre tid på webbläsaren. använder samma källkod som t.ex. Google Chrome. Chromium ska även vara bättre när det kommer till säkerhet och kodning, anses vara bättre för de som är mer är snäppet högre än average users.

SVG

Scalable vector graphics är ett XML baserat vektorformat för 2d bilder vilket gör att de kan animeras och det tillåter även att interaktivitet med bilderna.

PNG24

är ett format som optimerar bilder och tillåter användning av 24 bitars färg.Kan hantera mer färger och större filformat jämfört med t.ex. gif och png8. Upplösningen blir även bättre.
